ERA Group ERP Software Overview

Overview

The ERP software facilitates the operational flow of the ERA Group by managing client profiles, work orders, inspections, payments, and communication among three user types: Field Worker, Office Admin, and Client.

User Roles and Responsibilities

1. Client

  • Provides property details.
  • Receives inspection updates and invoices.
  • Approves and pays for services.

2. Office Admin

  • Creates client profiles and work orders.
  • Assigns work orders to field workers.
  • Manages invoices and financial transactions.

3. Field Worker

  • Inspects properties and documents issues.
  • Submits inspection reports, images, and documents.
  • Completes assigned tasks and updates progress.

Core Workflow

  1. Client Profile Creation: The admin creates a profile for a new client in the system.
  2. Work Order Creation: The admin generates a work order when the client places an order. The work order includes property details, service requirements, and deadlines.
  3. Work Order Assignment: The admin assigns the work order to a field worker.
  4. Inspection and Reporting: The field worker inspects the property and identifies issues. They upload images, documents, and inspection notes to the system.
  5. Invoice Management: The admin creates invoices for inspection fees and service charges. Clients receive the invoice for payment.
  6. Service Execution: The field worker addresses the identified issues. Progress is tracked using photos and job notes.
  7. Completion and Closure: After service completion, the admin updates the status to "Closed." Payment history and work details are recorded for future reference.

Key Pages and Features

1. General Page

Displays essential details like house location, work order status (e.g., "Growing," "Stopped," "Closed"), and other metadata.

2. Job Note Page

Allows field workers to log notes during the job, capturing new problems or observations made on-site.

3. Photo Page

Tracks work progress through images, including:

  • Before Work: Initial condition of the property.
  • During Work: Ongoing work images.
  • After Work: Final condition post-completion.

4. Invoice Page

Handles invoicing for both clients and contractors, recording payment history and outstanding balances.

5. Work Order List Page

Provides a dashboard for tracking and listing all work orders.

6. Create Work Order Page

Enables admins to create new work orders with detailed specifications.

7. Notification Page

Notifies users about new work orders, payment updates, and other alerts.

8. Waiting Payment Page

Tracks pending payments from clients, displaying work order details, tasks, and payment history.

9. Control Panel

Centralized management for feeds, photo tracking, and other system settings.

10. Profile Page

Manages user profiles and related settings.

System Benefits

  • Transparency: Detailed tracking of work progress and financial transactions.
  • Efficiency: Streamlined workflow for assigning, monitoring, and completing tasks.
  • Accountability: Photo tracking ensures accurate reporting and reduces dishonesty.
  • Client Engagement: Notifications and updates keep clients informed throughout the process.
